# Write File Utility

## Usage
Use `run_local_script` to execute `write_file.py`.

**Parameters**:
- `script_name`: "write_file.py"
- `args`: ["path/to/file", "content_string"]

**Example**:
```python
run_local_script(script_name="write_file.py", args=["output.txt", "Hello World"])
```
